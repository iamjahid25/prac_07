# CP1404 Practical- week 7

## About This Repository

This repository contains my solutions to the CP1404 Practical exercises for JCU's  Programming II.
I am a passionate and growing developer who is committed to writing clean, readable, and maintainable code.  
This repository showcases my progress, learning outcomes, and application of clean coding principles throughout the subject.

## Key Lessons Learned About Clean Code

- **Consistent Naming**: I learned the importance of using meaningful and consistent variable, function, and class names to make the code more readable and understandable.
- **Code Simplicity**: Writing simple, modular functions is more maintainable and easier to debug than trying to handle everything in one place.
- **Avoiding Repetition**: I’ve developed a habit of applying the DRY (Don't Repeat Yourself) principle to eliminate redundant code and improve maintainability.
